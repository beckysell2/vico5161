Heading Level 3 - Lists and Phrase Elements

These are phrase elements. They add structorial information to text elements.

  1. abbr: This identifies text as an abbreviation, such as FAQ.
  2. b: Text that has no extra importance but is style in bold font uses bold text phrase element.
  3. cite: Identifies a citation or reference, usually displayed in italics, such as "HTML and CSS: design and build websites" by Jon Duckett.
  4. code: Is used to be able to display code samples.
  5. dfn: Identifies a definition of a word or term, usually displayed in italics.
  6. em: Causes text to be emphasized; usually displayed in italics.
  7. i: Text that has no extra importance but is styled in italics.
  8. kbd: Identifies user text to be typed.
  9. mark: Text that is highlighted in order to be easily referenced.
  10. samp: Shows sample output.
  11. small: Legal disclaimer notices, such as fine print.
  12. strong: Strong importance.
  13. sub: Displays a subscript as small text below the baseline.
  14. sup: Displays a superscript as small text above the baseline.
  15. var: Identifies and displays a variable output.
